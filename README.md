# E1I5_project
중간평가

동기: 학교 도서관 관리 프로그램 개선을 위해서 시작한 프로젝트 가장 필요한 것: 건의 사항을 추가할 수 있는 기능 추가
가장 필요한 것: 자리만 맡아놓고 자리에 없는 사람들을 어떻게 하면 좋을까?


자료실
1. 책 리스트 입력
2. 책 대출
3. 책 반납
4. 책 검색
5. 책 정렬

열람실 관리
- 익명의 채팅으로 관리자에게 실시간으로  불만사항 전송 <- 핵심
- 열람실 몇시간 사용할지 몇시간 사용할지 미리 전송
- 30분정도 전에 연장 신청 가능
- 열람실 자리 현황파악
- 열람실 자리 신청 기능

회의실 기능
- 회의실 대여
- 예약

회원가입 (보류)
